DESCRIPTION

I built the Iowa Review website on Drupal 7. We needed to have data work in multiple ways on several different pages, so Drupal’s backend was the right fit. We also wanted a bold front page — full width, big hero image — so I used the base Adaptive Theme for mobility and designed up from there.
