DESCRIPTION

The International Writing Program asked for website redesign to help organize their great array of programs and offerings. The main design goal of the site was to increase usability and user navigation, as well as move the site into mobile. I built a wireframe using Bootstrap 3 and integrated it into their existing Drupal 7 system.

PROJECT DETAILS

  • TASK: Web Design, Responsive Design Branding, Graphic Design
  • DATE: June 2016
